Были изучены самые популярные нейронные сети по работе с изображениями и методы их оптимизации.
Также я научился работать с большими массивами данных при помощи библиотек numpy и pandas.
Еще были изучены способы визуализации данных с помощью библиотек matplotlib и seaborn.
